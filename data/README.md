## Storage Rules

- Do not commit credentials or access tokens.
- Do not commit private, sensitive, or very large data unless the project has
  explicitly decided that this is safe.
- If data lives outside the repository, record the path, access requirements,
  version/date, and contact person here.

## Provenance Questions

- Where did the data come from?
- Who is allowed to access it?
- What version or date does this copy represent?
- What preprocessing has already happened?
- Which files are canonical, and which are scratch or obsolete?
